== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for "CF7 File Download".
3. Click install.
4. Click activate.
5. Once the plugin is installed then you can see CF7 File Download on the left navigation bar of WordPress Dashboard.
6. Set the contact form id, the file url and the filename to use.
7. Save

== Description ==

CF7 File Download adds a file download functionality to Contact Form 7 after the form is submitted. It has a settings page where you can add the contact form id to trigger the download, the dowloaded file url and the downloaded file filename.
